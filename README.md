# Vacation-Biz-UI

度假 B 端 UI 组件库

## 开发命令

```shell
# install
npm install

# develop
npm run dev

# build
npm run build
```

## 目录结构说明

- `components/` 目录包含 `react` 组件代码，是最终输出给用户用的源码部分
- `lib/` 目录包含 `components/` 组件的编译结果，是最终输出给用户用的编译产物
- `src/` 目录包含文档示例代码，是开发阶段测试、验证和查看组件渲染效果的地方
- `dist/` 目录包含 `src/` 目录的产出，是部署为文档或者 `playground` 的目录
